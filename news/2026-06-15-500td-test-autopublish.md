# 500TD — тест автоматической публикации

Это тестовый патчноут для проверки системы uNews.

Проверяется:

* поиск новых файлов в папке `news/`;
* защита от повторной публикации;
* отправка текста в Telegram-канал;
* запись опубликованного файла в `data/published.json`.

Короткий текст для Telegram:

500TD — тест автоматической публикации через uNews.

Этот пост проверяет, что GitHub Actions умеет находить новый патчноут в папке news и отправлять его в Telegram без ручного запуска локального скрипта.
